# Code Explanation

コードや構造を視覚的かつ平易に解説するコマンド。

## 🎯 Objective

<goal>
ユーザーが指定したコード、またはプロジェクト全体のアーキテクチャを、Mentor の視点でわかりやすく解説すること。
</goal>

## 📋 Instructions

1. **Analyze Context:** コードの意図と、オセロ/強化学習における役割を分析する。
2. **Visualize:** `Mermaid` 記法を用いて、フローチャートやクラス図を作成する。
3. **Simplify:** 専門用語には補足を入れ、直感的な言葉で説明する。

## 🚫 Constraints

<rules>
- 単なる行ごとの翻訳（「これは変数宣言です」等）は禁止。
- 「なぜそのアルゴリズムを採用したか」という背景（Context）を含めること。
</rules>

## 📤 Output Format

1. **概要:** 3行要約。
2. **図解:** Mermaid ブロック。
3. **詳細解説:** ポイントを絞った説明。
