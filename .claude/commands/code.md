# Feature Implementation

指定された機能や修正をコードに落とし込むコマンド。

## 🎯 Objective

<goal>
ML Engineer の視点で、高品質かつ実行可能なPythonコードを実装すること。
</goal>

## 🚫 Constraints

<rules>
1. **Preserve Context:** 既存のコードスタイル（インデント、命名規則）を維持すること。
2. **Type Safety:** 全ての関数引数と戻り値に型ヒント (`Type Hint`) を付与すること。
3. **Docstring:** Google Style の Docstring を記述すること。
4. **Vectorization:** `for`ループの代わりに `numpy` の演算を使用すること（可能な場合）。
</rules>

## 📋 Instructions

1. 編集対象のファイルを読み込む。
2. `<rules>` に従ってコードを修正・作成する。
3. シンタックスエラーがないか確認する。

## 📤 Output Format

変更したファイルのコードブロックを提示してください。
