# Bug Fix

エラーやバグを修正するコマンド。

## 🎯 Objective

<goal>
提供されたエラースタックトレースや不具合報告に基づき、根本原因を解決すること。
</goal>

## 📋 Instructions

1. **Analyze Error:**
   - エラーメッセージと発生箇所を特定する。
   - 必要であれば `print` デバッグやログ出力を提案する。
2. **Root Cause Analysis:**
   - 「なぜその値が入ったのか」という根本原因を推論する。
3. **Apply Fix:**
   - 対症療法（`try-catch` で握りつぶす等）ではなく、ロジック修正を行う。

## 📤 Output Format

- **原因:** 何が悪かったのかの簡潔な説明。
- **修正:** 修正後のコードブロック。
